API Usage Example

Currently only supports 'Singapore' and 'Pima Arizona US'


Request summary data for a country (JSON):

Format: https://covid-case-dashboard.herokuapp.com/api/<country>
# curl "https://covid-case-dashboard.herokuapp.com/api/Singapore"

Request summary data for a country, province or state, and county (JSON):

Format: https://covid-case-dashboard.herokuapp.com/api/<country>/<province_state>/<county>
# curl "https://covid-case-dashboard.herokuapp.com/api/US/Arizona/Pima"
