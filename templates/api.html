API Usage Example

Request summary data for a country (JSON):

Format: {{ base_url }}/<:country>/latest
# curl "{{ base_url }}/Singapore/latest"

Request summary data for a country and province or state (JSON):

Format: {{ base_url }}/<:country>/<:province_state>/latest
# curl "{{ base_url }}/Australia/Queensland/latest"

Request summary data for a country, province or state, and county (JSON):

Format: {{ base_url }}/<:country>/<:province_state>/<:county>/latest
# curl "{{ base_url }}/US/Arizona/Pima/latest"

Request daily-case data for a country (JSON):

Format: {{ base_url }}/<:country>
# curl "{{ base_url }}/Singapore"

Request daily-case data for a country, province or state (JSON):

Format: {{ base_url }}/<:country>/<:province_state>
# curl "{{ base_url }}/Australia/Queensland"

Request daily-case data for a country, province or state, and county (JSON):

Format: {{ base_url }}/<:country>/<:province_state>/<:county>
# curl "{{ base_url }}/US/Arizona/Pima"
